Welcome to my blog. Posts are written in Markdown and live in `src/content/posts/`.

To add a new post, create a `.md` file in that folder with the following frontmatter:

```markdown
---
title: "Your Post Title"
publishedOn: 2024-06-15
---

Your content here...
```

The filename (without `.md`) becomes the URL slug — e.g. `my-post.md` → `/post/my-post`.

To keep a draft, move the file to `src/content/drafts/`. It will never be built or published.
